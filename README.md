# 프로젝트 소개
스프링 부트를 이용한 쇼핑몰 웹 프로젝트(개인 프로젝트)<br/><br/>
### 기능 정리
- 회원 기능
    - 로그인, 회원가입, 마이페이지(회원 정보 조회, 배송지 수정), 회원 리스트 조회(관리자 계정만 가능)
- 게시물 기능
    - 공지사항 및 매거진 등록(관리자 계정만 가능), 조회
- 상품 기능
    - 상품 등록(관리자 계정만 가능), 상품 검색 및 조회
- 주문 기능
    - 주문, 주문 조회, 주문 취소
      <br/><br/>
